##vue 安装
```
vue init webpack my-project
```

安装主题工具
```
安装工具:
npm i element-theme -D

安装主题:
npm i element-theme
npm i element-theme-chalk -D
or
npm i https://github.com/ElementUI/theme-chalk -D
产生theme变量配置文件: 
node_modules/.bin/et -i
产生theme目录:
node_modules/.bin/et 
```

```
安装element-ui:
npm i element-ui -S
```

```
npm install
npm run dev

```
